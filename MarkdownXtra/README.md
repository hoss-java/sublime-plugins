# Sublime-text plugins - MarkdownXtra 
Enhancing Sublime-text to work with markdown files

- Package - MarkdownXtra
- Requiments - None

## Install
Copy the package (MarkdownXtra) to `Packages/` (On Linux `~/.config/sublime-text/Packages/`)

# How to use
The Markdown eXtra automatically predicts coming lines prefixes and adds them when a new line is added!
