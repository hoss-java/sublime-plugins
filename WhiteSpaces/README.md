# Sublime-text plugins - Whitespaces
Enhancing Sublime-text to manage whitespaces.

- Package - Whitespaces
- Requiments - None

## Install
Copy the package (Whitespaces) to `Packages/` (On Linux `~/.config/sublime-text/Packages/`)

# How to use
The plugin improves internal sublime-text whitespace functionalities and also adds a new feature to highlight white spaces.

To modify keys `Packages/WhiteSpaces/Default (Linux).sublime-keymap`

The default added hotkeys:
* ctrl+alt+W Toggle between internal whitespace, display modes, **selection**, **all** and **none**
* ctrl+alt+H Toggle whitespace highlighting ON/OFF

WhiteSpaces adds also two new items to the context menu (right click).
