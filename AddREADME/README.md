# Sublime-text plugins - Add README
Add an new item to Side bar menu to added README.md file to a project based on templates

- Package - AddREADME
- Requiments - None

## Install
Copy the package (AddREADME) to `Packages/` (On Linux `~/.config/sublime-text/Packages/`)

# How to use
AddREADME adds a menu to the sidebar menu (right click) ,'Add README', if the selecteted path has not already 'README.md'.

To Add new templates copy templates files to `Packages/AddREADME/templates`
